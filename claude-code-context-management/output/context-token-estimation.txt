============================================================
Claude Code autocompact 触发阈值的 headroom 分析
============================================================
上下文窗口:          200,000 tokens
触发阈值 (83.5%):    167,000 tokens
headroom (剩余):      33,000 tokens
声称的 buffer:        33,000 tokens
buffer 与 headroom 差值:     0 tokens

  → 33K buffer 与 (1-83.5%)*200K = 33,000 高度一致
  → 这意味着 Anthropic 把 buffer 设计成 'context 窗口剩余空间'

============================================================
会话累积场景：何时触发 autocompact?
============================================================
假设：每轮对话约 800 个汉字 + 200 字符英文/代码
每轮估算 tokens:  583
触发阈值 tokens:  167,000
约 286 轮对话后触发 autocompact

  → 这意味着重度使用一天，autocompact 几乎一定触发
  → 用户感知到的'用着用着就忘'，正是这一刻发生的
